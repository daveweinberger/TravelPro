## Design Notes

### Visual Hierarchy
- **Top Level**: Unified view of all trips with tag-based filtering
- **Primary Actions**: Large, accessible buttons for key functions
- **Information Density**: Balanced - detailed enough for power users, clean enough for quick scanning
- **Status Indicators**: Clear use of icons and color-coded warnings (âš ï¸ for gaps, âœ“ for complete)

### Key Design Decisions

1. **Unified Multi-Trip Organization**: 
   - **Single calendar/list view** shows all trips together
   - **Tag system** allows users to categorize and filter trips (e.g., "Vacation", "Business", "Europe", "Asia")
   - **Trip badges** (country flags, emojis) provide visual differentiation on calendar
   - **Filter controls** enable quick focus on specific trip types or tags
   - Users can view all trips simultaneously or filter by tags

2. **Enhanced Gap Detection Display**:
   - **Inline calendar alerts**: Gaps marked with âš ï¸ symbol directly on calendar dates
   - **Calendar day details**: Gaps shown within the event timeline when day is selected
   - **Dedicated Notifications View**: Centralized [ğŸ”” Alerts] button leads to comprehensive gap overview
   - **Priority levels**: Gaps categorized as High/Medium/Low priority based on duration and context
   - **Smart notifications**: Pre-trip reminders and other important alerts in unified alerts screen
   - **Visual timeline**: Gap detail screens show clear before/after event context

3. **Tag System Implementation**:
   - Tags displayed as colored badges on trip items
   - Quick filter bar at top for easy trip segmentation
   - "Manage Tags" button for creating custom tags
   - Tags help organize trips by: Type (Vacation/Business), Region (Europe/Asia), Purpose (Work/Leisure)

4. **Import Flow**:
   - Three clear options: Upload, Email, Manual entry
   - Processing feedback with progress indicators
   - Review step prevents incorrect data entry
   - Smart insights provided immediately after import
   - Automatically detects gaps during import process

5. **Recommendations System**:
   - Multiple filters: For You, Gaps, Location, Date
   - Context-aware suggestions based on user interests
   - Pre-trip and during-trip sections
   - Integration with gap-filling functionality
   - Location-aware suggestions based on current/upcoming destinations

6. **Accessibility**:
   - High contrast elements
   - Clear button labels
   - Consistent iconography
   - Dark mode support for different lighting conditions
   - Notification badges for quick status awareness

### Component Patterns
- Cards for grouping related information
- Progressive disclosure (View All, Details buttons)
- Inline actions for quick operations
- Confirmation screens for data validation
- Tag badges for visual categorization
- Alert counters for awareness without interruption
